```
Create a 3D rendered chibi robot character in DreamWorks/Pixar vinyl toy style. Dark moody cyberpunk background. Character is small, stocky, cute but powerful proportions like a Funko Pop figure.

Character: [CHARACTER NAME]
Color scheme: [COLOR DETAILS]
Key features: [UNIQUE FEATURES]
Accessories: [WEAPONS/TOOLS/ITEMS]
Mood: [MOOD]
Lighting: [GLOW COLOR] neon glow effects on armor edges and eyes
Style: High quality 3D render, dramatic lighting, cinematic composition
```

## MASTER CARD TEMPLATE PROMPT (after you have the render)

Once you have the 3D render, paste this into Gemini with the render attached:

```
Create a horizontal trading card layout using this character image. Match this exact format:

Top left: [ZONE] in brackets, uppercase
Below zone: Character name in large bold text
Below name: [ROLE] in brackets
Character image takes up left ~40% of the card
Bottom left under image: stat bars in a row:
  IQ: [VALUE] SPEED: [VALUE] POWER: [VALUE] TRUST: [VALUE] SOUL: [VALUE]
  Stats with filled bar indicators proportional to value (max 100)

Right side top: [ORIGIN] section with backstory text in clean paragraph
Right side bottom: [MEGA ENHANCEMENTS] section with LOCKED icon, listing enhancement names
Bottom right corner: QR code placeholder, "theangel.com/card/[name]", Card #[NUMBER]

Color scheme: dark background, cyan/blue accent borders, monospace/terminal style fonts
Style: cyberpunk trading card, clean readable layout, high contrast
```
